# 🌿 Botanical Calculator

A Pinterest-inspired calculator with a soft floral aesthetic, built using HTML, CSS, and JavaScript.

## ✨ Features
- **Day/Night Modes:** Switch between light watercolor vibes and dark moody florals.
- **Glass-morphism:** A frosted glass effect on the calculator body.
- **Fully Functional:** Basic math operations (addition, subtraction, etc.).

## 🎨 Theme
- **Day Mode:** Cream & Sage Olive (from `image_8ea965.jpg`)
- **Night Mode:** Deep Forest & Charcoal (from `image_8efb5f.jpg`)

## 🚀 Built With
- HTML5
- CSS Grid & Variables
- Vanilla JavaScript
